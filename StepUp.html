StepUp

Event Category: Autonomous Robotics

Problem Statement:

Build an autonomous robot that is capable of traversing a grid and arranging the blocks placed on nodes in the increasing order of height.

Prizes Worth: Rs.46,000

Task

The autonomous robot has to traverse a grid and should be able to sort wooden blocks placed randomly according to their height (in ascending order) along the middle line.

Arena

Unsorted Arena Round 1

  1. The arena for Round 1 is a black plywood surface (Dimensions to be put up soon) .  This would enclose a 7x5 grid.
  2. There will be 3 blocks on the grid placed haphazardly.
  3. The size of each grid square is 30 cm. The width of every line is 3 cm.
  4. There are a maximum of 7 grid nodes in each row.
  5. All dimensions have a tolerance of 10%.

Unsorted Arena Round 2

 

 

  1. The arena for Round 2 is a black plywood surface ( Dimensions to be put up soon).  This would enclose a 7x5 grid.
  2. There will be 5 blocks placed haphazardly on the grid.
  3. The size of each grid square is 30 cm. The width of every line is 3 cm.
  4. There are a maximum of 7 grid nodes in each row.
  5. All dimensions have a tolerance of 10%.

Unsorted Arena Round 3

  1. The arena for Round 3 is a black plywood surface ( Dimensions to be put up soon).  This would enclose a 7x3 grid.
  2. There will be 5 blocks placed haphazardly on the grid.
  3. The size of each grid square is 30 cm. The width of every line is 3 cm.
  4. There are a maximum of 7 grid nodes in each row.
  5. All dimensions have a tolerance of 10%.

Rounds

         Round 1 : 

  • There will be a single run of 4 minutes.
  • The size of the grid will be 7x5.
  • In this round, there will be 3 blocks placed on the grid randomly.
  • The task of the robot is to sort the blocks in ascending order of height such that all the blocks lie in the middle 3 nodes ( out of 7 ) in the middle line.
  • The smallest block should be the closest to the start zone.
  • The robot has to indicate a number showing block position according to its height (using 5 leds or display the number on lcd if participant is using one) while picking it. Points will be awarded for this. The mode of display is to be told to the organizer before the begin of the run.

         Round 2:

  • There will be a single run of 6 minutes.
  • The size of the grid will be 7x5.
  • The 5 blocks will be distributed randomly all over the grid.
  • The robot has to detect each of these blocks, and then place them in the middle line (again in the correct position ) in increasing order of height.
  • The smallest block should be the closest to the start zone.
  • The robot has to indicate a number showing block position according to its height (using 5 leds or display the number on lcd if participant is using one) while picking it. Points will be awarded for this. The mode of display is to be told to the organizer before the begin of the run.

 Round 3:

  • There will be a single run of 8 minutes.
  • The size of the grid will be 7x3.
  • The 5 blocks will be distributed randomly all over the grid.
  • The robot has to detect each of these blocks, and then place them in the middle line (again in the correct position ) in increasing order of height.
  • The smallest block should be the closest to the start zone.
  • The robot has to indicate a number showing block position according to its height (using 5 leds or display the number on lcd if participant is using one) while picking it. Points will be awarded for this. The mode of display is to be told to the organizer before the begin of the run.

GENERAL RULES :

  1. The robot should be completely autonomous.
  2. All arena dimensions have a tolerance of 10%.
  3. Each team can consist of a maximum of 4 members.
  4. Each team should have unique participants i.e. no two teams can have even a single participant common.
  5. The team members can be from different institutes or colleges.
  6. The right spirit of participation is expected from the participants.
  7. The decision of the Team ROBOTIX will be final and binding.

ROBOT

  1. The robot should be completely autonomous.
  2. The robot needs to traverse a 7x5 grid in Round 1 and Round 2 and 7x3 grid in Round 3.
  3. The grid is made up of white strips on a black background as the lower surface.

 

BLOCKS

  1. The blocks will be made of the same material, their base size will be 10 cm * 10 cm.
  2. Height of each block will be different as shown in figure.
  3. The initial arrangement of the blocks on the grid cannot be hardcoded. The robot must traverse the arena and find the positions and detect the heights of the blocks autonomously.
  4. Every movement of a block from one node to the next one is considered as a step.
  5. After the sorting is done , the smallest block should lie closest to the START zone. This orientation will be fixed for each participant.
  6. The dimensions of the blocks are subject to change.

 

TRAVERSAL

  1. For both the rounds, the starting position of the robot is at (0, 0) position of the grid.
  2. The robot has to begin its run facing in the y direction.

 

RESTARTS AND TIME-OUTS

  1. A maximum of 2 Timeouts of 2 minutes each may be taken. Penalty will be awarded for each timeout and robot will start from the last node crossed.
  2. The participant's robots can have a maximum of 3 restarts. A penalty will be imposed on the team for every restart that they take.
  3. In case of a restart the participant's robots will be set to their initial positions. Timer will be set to zero and the run will start afresh with the addition of the penalty for restart.
  4. If the robot goes off track or outside the arena, then the team has to take a restart or call off their run.

The arena and the dimensions are tentative as of now.

  1. Each robot can have maximum dimension of 30*25*30 cm3 (L*B*H) respectively.
  2. No part/mechanism of/on the robot should exceed the given dimensions before the commencement of the event run. The robots can exceed their respective dimensions once the event commences.
  3. The robots need to follow lines and traverse only along the grid.
  4. The autonomous robots should be on-board processing robots, i.e., the robots cannot be controlled by a remotely kept computer.
  5. LEGO kits or its spare parts or pre-made mechanical parts are not allowed. (http://en.wikipedia.org/wiki/Lego_Mindstorms)
  6. Ready-made gearboxes, sensors, development boards can be used but no other part of the robot should contain any ready-made components. Simple car bases with no extra features may be used.
  7. The bots should not harm the StepUp event arena in any way. If it does so, a penalty will be imposed on the team. The magnitude of the penalty will be decided by Team ROBOTIX.
  8. Processors of less than 16-bits are allowed. ARM processors are not allowed.
  9. Ready-made gripper is also allowed but it should be controlled autonomously.

Scoring

  • Initial base score for Round 1 and Round 2(B) : 1000 points
  • For detecting a block and lighting an led in response(D)  :  +75
  • For each correctly placed block(C) : +150

             ( Correctly placing a block means the block is placed at the specified position )

  • For number of moves made by the bot with the block(NM) : -20
  • For collisions made with other blocks when one block being arranged(NC) : - 75
  • Timeouts(T) : -50 points
  • Restarts(R)  : -80 points
  • Time Bonus(TB) : (AT - RT)/3   (where AT is the maximum allowed run-time and RT is the actual run-time)

Scoring Formula : B + 75*D + 150*C - 20*NM - 75*NC - 50*T - 80*R + TB
